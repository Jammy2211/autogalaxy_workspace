The ``results/examples`` folder contains example scripts for using the results of a **PyAutoGalaxy** model-fit.

Files
-----

- ``samples``: Non-linear search model results (parameter estimates, errors, etc.).
- ``samples_via_aggregator``: Loading samples via the aggregator.
- ``queries``: Query the database to get certain modeling results (e.g. all models where `effective_radius > 1.0`).
- ``models``: Inspect the models in the database (e.g. visualize the galaxy images).
- ``data_fitting``: Inspect the data-fitting results in the database (e.g. visualize the residuals).
- ``galaxies_fit``:  Inspecting individual galaxies (e.g. their light profiles) and fits (e.g. model-images, chi-squareds, likelihoods).
- ``interferometer``: Analysing the results of an interferometer fit.
